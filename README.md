Global store utility for an async operation lifecycle and chain of callbacks.

Ever had a situation where you needed to pass a value down to a series of callbacks/promises without polluting the function signature? Async Store might be the solution you need. When you set values in Async Store, those values are accessible until all functions called from the original function ‚Äì synchronously or asynchronously ‚Äì have finished executing.
